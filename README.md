                <th rowspan="1">Jobsheet 12</th>
                <td>Membahas tentang Tree Traversal, yaitu proses mengunjungi dan mendata setiap simpul dalam sebuah struktur data tree. Ada tiga cara untuk melakukan Tree Traversal, yaitu Inorder Traversal, Preorder Traversal, dan Postorder Traversal. Inorder Traversal melintasi tree dengan cara mengunjungi semua simpul di sebelah kiri, kemudian root node, dan akhirnya simpul di sebelah kanan. Preorder Traversal melintasi tree dengan cara mengunjungi root node, kemudian simpul di sebelah kiri, dan akhirnya simpul di sebelah kanan. Postorder Traversal melintasi tree dengan cara mengunjungi simpul di sebelah kiri, kemudian simpul di sebelah kanan, dan akhirnya root node. Implementasi Tree Traversal dilakukan dengan menggunakan struktur data tree yang terdiri dari simpul dan sub-tree. Proses pengujian dilakukan dengan menggunakan fungsi-fungsi seperti inorder, preorder, dan postorder untuk mengunjungi simpul dalam tree. Contoh implementasi Tree Traversal juga disajikan dalam jobsheet ini, termasuk contoh penggunaan fungsi insert untuk menambahkan data ke dalam tree dan fungsi search untuk mencari data dalam tree.</td>
